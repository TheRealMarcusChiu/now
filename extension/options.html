Life Tracker

Settings for the tracker. Edits here sync live with the toolbar popup.

Server endpoint

Where logged events are sent — the Node server from your repo (node server/server.mjs). Leave empty to use the same origin the site is served from.

Ignored domains

One entry per line — an exact host or a glob pattern. These are never logged.

* matches anything — *.lan ignores every host ending in .lan; login.* ignores every login.-prefixed host. Blank lines and # comments are ignored.

Ignored URLs

One full URL per line. More specific than a domain — ignores a single page while the rest of the site keeps logging.

A trailing / and #fragments are ignored when matching, so https://youtube.com and https://youtube.com/ are the same.
